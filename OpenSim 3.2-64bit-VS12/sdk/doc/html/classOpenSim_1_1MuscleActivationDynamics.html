Detailed Description

An abstract class for modeling muscle activation dynamics.

Activation models that derive from this base class are responsible for creating and maintaining whatever states and/or cache variables they require to override the getActivation() and setActivation() pure virtual methods.

Properties

  • minimum_activation: Smallest permitted activation value.
  • maximum_activation: Largest permitted activation value.
  • default_activation: Default activation value.

Conditions 

0 <= minimum_activation <= default_activation <= maximum_activation <= 1

Default Property Values 

minimum_activation .... 0
maximum_activation .... 1
default_activation .... 0.5
Author
Thomas Uchida, Ajay Seth, Michael Sherman
